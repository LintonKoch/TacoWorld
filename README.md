# TacoWorld
A local web site used to order Tacos. This branch is showing a local h2 database, 
but utilizing Spring Data JDBC instead of JDBC.

HTML <br/>
CSS  <br/>
Javascript  <br/>
Java  <br/>

spring-boot-starter-web <br/>
spring-boot-devtools <br/>
